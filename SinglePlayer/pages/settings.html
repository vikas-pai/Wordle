Welcome to Wordle, the exciting word puzzle game that challenges your vocabulary and creativity! In this game, you have to guess the word.

You have to type in a word which is present in the wordlist.
If the word is present there will be any of the following outcomes to the letter:

-The letter will be green if the letter exists and is in the correct place.

-The letter will be yellow if the letter exists and isn't in the correct place.

-The letter will be gray if the letter doesn't exist.

Software and Platforms:

Wordle was developed using HTML5,CSS and JavaScript, and is available on all major web browsers and is compatible only on laptops.
